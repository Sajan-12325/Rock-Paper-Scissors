# Rock, Paper, Scissors


This is a simple command-line Rock, Paper, Scissors game implemented in Python. The game allows a user to play against the computer for five rounds. The computer randomly selects a choice from Rock, Paper, or Scissors, and the winner of each round is determined based on the standard game rules.


GAME RULES

Rock beats Scissors

Scissors beat Paper

Paper beats Rock

HOW TO PLAY

Run the Python script.

Enter your choice (Rock, Paper, or Scissors).

The computer will randomly select its choice.

The winner of the round is announced.

The game continues for five rounds.

After all rounds are completed, the final winner is displayed based on the scores.
